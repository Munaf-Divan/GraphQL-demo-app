# How to run the app

- Clone the project
## Backend App

- Install dependencies
  ```
    npm install
  ```
- Create `.env` file with variables mentioned in `.env.example`.
  - Create `postgresql` database and save connection url in .env file.
- Create and Run migration.
  ```
  npm run migrate
  ```
- Run the project
  ```
  npm run dev
  ```
## Frontend App

- Install dependencies
  ```
    npm install
  ```
- Create `.env` file with variables mentioned in `.env.example`.
- Run the app
  ```
    npm run dev
  ```
